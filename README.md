# GeoBuddy

A very simple SwiftUI app created as university assignment.

Allows for easy conversion between the 3 most popular geographic coordinates
formats:

- (DMS) Degrees Minutes Seconds
- (DM) Decimal Minutes
- (DD) Decimal Degrees
